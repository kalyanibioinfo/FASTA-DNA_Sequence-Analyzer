FASTA DNA Sequence Analyzer

Author

Kalyani Bedkihalkar

---

Project Overview

This Python project reads DNA sequences from a FASTA file and performs basic bioinformatics analysis on each sequence.

---

Features

- Read DNA sequences from a FASTA file
- Calculate sequence length
- Calculate GC content (%)
- Count nucleotides (A, T, G, C)
- Generate complementary DNA sequence
- Generate reverse complement sequence
- Convert DNA to RNA (Transcription)
- Translate RNA into a protein sequence using the standard genetic code
- Display a final project summary

---

Project Workflow

1. Read FASTA file
2. Identify gene names
3. Read DNA sequences
4. Calculate sequence length
5. Calculate GC content
6. Count nucleotides
7. Generate complement sequence
8. Generate reverse complement
9. Transcribe DNA into RNA
10. Translate RNA into protein
11. Display project summary

---

Input File

The program expects a FASTA file named:

"fasta.txt"

Example:

>Gene1
ATGCGATCG

>Gene2
AACCGGTTAA

---

Output

For each gene, the program displays:

- Gene Name
- DNA Sequence
- Sequence Length
- GC Content
- Nucleotide Counts
- Complement Sequence
- Reverse Complement
- RNA Sequence
- Protein Sequence

Finally, it displays:

- Total Genes
- Average GC Content
- Longest Gene Length
- Shortest Gene Length
